# 🤖 PyGui MT4 Agentic AI 실습: HRI 제어 시스템 복구 미션

## 📝 프로젝트 개요
이 리포지토리는 MT4 로봇 팔과 Unity 기반의 가상 환경을 연동하여 제어하는 시스템의 베이스 코드입니다. 현재 시스템의 핵심 제어 로직이 의도적으로 삭제되어 있습니다. 
여러분의 목표는 **생성형 AI(Agentic AI)를 활용하여 누락된 코드를 완벽하게 복구하고 시스템을 정상화**하는 것입니다.

## 🎯 실습 미션 (Missions)
`Quiz_Code.py` 코드 내부의 `TODO` 주석을 확인하고, AI에게 적절한 프롬프트를 입력하여 다음 두 가지 핵심 노드를 복구해야 합니다.
